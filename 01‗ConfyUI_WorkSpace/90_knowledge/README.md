# 90_knowledge（ナレッジ / 引き継ぎ用）

このフォルダは **他のAI・他の担当者が「今どうなっているか」を最短で把握する**ための集約場所です。
（＝ナレッジベース / ハンドオフ資料 / SSOT: single source of truth）

## 使い方
- まず [現環境スナップショット.md](現環境スナップショット.md) を読めば全体像が分かります。
- 詳細は各一次情報（`00_Plan/` `10_manual/` `20_agents/` `40_prompts/` `50_batch/` `Report/`）へのポインタからたどってください。
- このリポジトリは GitHub: `SHUNSUKE-Ks/Portable-Comfy-Studio-` で共有されています。

## 注意
- ここは**読めば現状が分かる**ことを目的にした要約です。仕様の正本は各一次情報側。
- 実機ログ `Report/test_logs/` と巨大バイナリ（`ComfyUI/` の venv・models）は git 管理外。クローン先には含まれません（必要なら再生成・再取得）。
- 内容が古くなったら、変更した人が `現環境スナップショット.md` の日付と該当箇所を更新してください。
